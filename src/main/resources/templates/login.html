Welcome to SecureStack

SecureStack is a full-stack Kotlin Spring Boot application that helps development teams stay secure and up-to-date by monitoring software tools and project dependencies for version updates. It supports GitHub login, team-based organization, app-specific configuration, and sends email alerts when newer versions are available — making dependency tracking simple, proactive, and secure.
